MANUAL DE USUARIO
 TPM-QLabs - MANEJO DE EMPLEADOS


Cuando el usuario selecciona del menú principal la opción Usuarios, el sistema desplegiega una pantalla como la que se muestra en la Figura 1. En esta pantalla el usuario podrá ver los empleados de la empresa, registrar nuevos empleados, actualizar su información, habilitarlos y deshabilitarlos.
Figura 1. Pantalla para el manejo de empleados.
Agregar empleado
Para agregar un empleado pulse el botón Agregar (1), diligencie el formulario ubicado al lado derecho de la pantalla (2) y pulse el botón Crear (3), tal y como se muestra en la Figura 2.

Figura 2. Pantalla para agregar un empleado.

La información que se debe ingresar para el registro de un usuario es la siguiente:

    Datos del usuario
  • Login de usuario: Ingrese el login de usuario  (es un campo obligatorio).
  • Password: Ingrese el pasword de usuario el cual debe ser mínimo de 8 caracteres alfanumericos (es un campo obligatorio).
  • Repetir password: Repita el password (es un campo obligatorio).
  • Perfil: Seleccione un perfil de usuarios, los perfiles definen que puede hacer cada persona, (es un campo obligatorio)
    • Administrador: Administrador del sistema, dentro de las tareas principales que puede desempeñar este perfil están manejo de métodos, manejo de equipos , manejo de usuarios y empleados, otra característica importante de este usuario es la posibilidad de ver reportes que permitirán la toma de decisiones importantes para la empresa cliente.
    • Analista: Es el perfil de los empleados que pueden ingresar tiempos de corrida de sus análisis.
    • Coordinador o supervisor: Es un perfil para usuarios que requieran ingresar a los reportes de toma de decisiones.
  • Habilitado: Seleccione si el usuario está habilitado o deshabilitado, tenga en cuenta que los usuarios deshabilitados no podrán ingresar al sistema (es un campo obligatorio).
Nota: El login y el password deben ser entregados a los usuarios para que estos ingresen al sistema.

    Datos del empleado
  • Nombre: Ingrese el nombre del empleado  (es un campo obligatorio).
  • Apellidos: Ingrese los apellidos del empleado  (es un campo obligatorio).
  • Tipo identificación: Seleccione el tipo de identificación del empleado (es un campo obligatorio).
  • Número identificación:  Ingrese el número de identificación (es un campo obligatorio).
  • Empresa: Seleccione la empresa a la cual pertenece el empleado.
  • Escoger Foto: Utilícelo para subir al sistema una foto del usuario (es opcional)


Actualizar
Para actualizar un empleado, seleccione el empleado que desea actualizar de la lista (1), actualice la información en el formulario del lado derecho y pulse el botón Actualizar (2). El sistema desplegara una pequeña ventana donde le preguntará porque razón desea actualizar el empleado, digite la razón (3) y pulse el botón Aceptar (4), tal y como se muestra en la Figura 3. Luego el sistema le mostrará un breve mensaje en el que informará del éxito del cambio. 

En caso de que alguno de los usuarios haya olvidado la contraseña para el acceso al sistema, el administrador puede hacer uso de esta interfaz para asignarle una nueva contraseña.  Para asignar una nueva contraseña a un usuario, se debe seguir el mismo proceso para la actualización de información descrito en la Figura 3 con la única diferencia de que en vez de ingresar algún dato del usuario o empleado, se ingresa la nueva contraseña en los campos Password y Repassword (5).



Figura 3. Pantalla para actualizar la información de un empleado.


Deshabilitar

Para deshabilitar un empleado, seleccione un empleado de la lista y en el formulario del lado derecho seleccione en el campo Habilitado la opción No (1) y pulse el botón Actualizar (2), tal y como se muestra en la Figura 4.  El sistema desplegará una pequeña ventana donde le preguntará porque razón desea actualizarlo, digite la razón y pulse el botón Aceptar. Luego el sistema le mostrara un breve mensaje en el que informa del éxito de la actualización.  Además, recuerde que los usuarios deshabilitados no podrán ingresar al sistema.

En la Figura 2 también se muestra el listado de todos los usuarios del sistema, tanto los habilitados como los deshabilitados: los habilitados están acompañados dentro de la lista por una estrella amarilla mientras que los deshabilitados estan acompañados por una estrella blanca.  Usted puede útilizar los botones de la barra superior para ver solo los usuarios habilitados (3), los usuarios deshabilitados (4) o todos los usuarios (5).

Figura 4. Pantalla para deshabilitar un empleado.
