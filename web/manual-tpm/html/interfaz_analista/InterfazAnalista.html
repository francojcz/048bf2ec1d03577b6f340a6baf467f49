MANUAL DE USUARIO
 TPM-QLabs - INTERFAZ ANALISTA


Una vez hayamos digitado correctamente la clave y la contraseña, el usuario analista ingresará a una pantalla como se muestra en la Figura 1.

La interfaz del analista está compuesta de los siguientes elementos.

(1) Cabecera: Nombre y logo de la empresa.
(2) Barra informativa y de utilidades: Muestra los datos del operario, la máquina seleccionada, la fecha, el enlace para acceso a reportes, entre otros.
(3) Barra de botones: Contiene los botones para registrar nuevas corridas y manejar eventos.
(4) Barra de pestañas: Contiene las pestañas que dividien el área de trabajo en información de método, de muestras y de columnas.
(5) Área de trabajo: Área de trabajo para ingresar los datos de las corridas analíticas.
(6) Ayudas: Ayudas conceptuales sobre los indicadores.
(7) Gráfica de tiempos: Gráficos.
