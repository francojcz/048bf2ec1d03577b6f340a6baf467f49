Eliminar Certificados


MANUAL DE USUARIO
 TPM-QLabs - ELIMINAR CERTIFICADOS
Cuando el usuario selecciona la opción Eliminar certificados ubicada en el menú principal, el sistema desplegará una pantalla como la que se muestra en la Figura 1. En esta pantalla el usuario podrá eliminar certificados de computadores que actualmente no se encuentran en uso.

Para llevar a cabo el proceso de eliminación de un certificado, el usuario debe seleccionar un computador de la lista de computadores registrados (1) y pulsar el botón Eliminar certificado (2).  El sistema desplegará un mensaje de confirmación en el cual el usuario debe pulsar el botón Si (3) para dar continuación al proceso de eliminación del certificado.  Luego el sistema le mostrará un breve mensaje en el que le informará el éxito de la eliminación.
Figura 1. Pantalla para la eliminación de certificados.
