MANUAL DE USUARIO
 TPM-QLabs  - MANEJO DE ADMINISTRADORES


Cuando el usuario selecciona la opción Manejo de administradores ubicada en menú principal, el sistema le desplegará una pantalla como la que se muestra en la Figura 1.  En esta pantalla el usuario podrá registrar, eliminar y actualizar la información de los usuarios con perfil administrador, además de asociarlos con las empresas previamente registradas.
Figura 1. Pantalla de manejo de empleados

Agregar administrador

Para agregar un usuario administrador pulse el botón Agregar, diligencie el formulario ubicado al lado derecho de la interfaz para el manejo de administradores y pulse el botón Guardar.  Los datos del usuario que se deben ingresar para su registro se describen a continuación:
  • Login de usuario: Ingrese el login de usuario  (es un campo obligatorio).
  • Password: Ingrese el password del usuario el cual debe tener como mínimo 8 caracteres (es un campo obligatorio).
  • Repetir password:  Repita el password (es un campo obligatorio).
  • Perfil:  Seleccione un perfil de usuario.  Un perfil de usuario define las funcionalidades a las cuales tiene acceso cada persona.  Para esta interfaz, el único perfil disponible es el perfil Administrador el cual tiene asignado una serie de tareas dentro de las cuales se encuentran: manejo de métodos, manejo de equipos, manejo de empleados, además de tener la posibilidad de visualizar los reportes que permitirán la toma de decisiones importantes para la empresa cliente (es un campo obligatorio).
  • Habilitado: Seleccione si el usuario está habilitado o deshabilitado.  Tenga en cuenta que los usuarios deshabilitados no podrán ingresar al sistema (es un campo obligatorio).
Nota: El login y el password deben ser entregados a los usuarios con perfil administrador para que estos ingresen al sistema.

 También se deben ingresar los siguientes datos del empleado:
  • Nombre: Ingrese el nombre del empleado (es un campo obligatorio).
  • Apellidos: Ingrese los apellidos del empleado (es un campo obligatorio).
  • Tipo identificación: Seleccione el tipo de identificación del empleado (es un campo obligatorio).
  • Número identificación: Ingrese el número de identificación (es un campo obligatorio).
  • Empresa: Seleccione la empresa a la cual pertenece el empleado, la cual debe haber sido registrada previamente en el sistema.
  • Escoger foto: Utilícelo para subir al sistema una foto del usuario (es opcional)


Actualizar

Para actualizar la información de un usuario, seleccione el empleado que desea actualizar de la lista de usuarios registrados, actualice la información del usuario haciendo uso del formulario ubicado al lado derecho de la interfaz para el manejo de administradores y pulse el botón Actualizar.  El sistema desplegará una pequeña ventana donde debe ingresar la razón de la actualización de la información del usuario.  Una vez ingresada dicha razón, debe pulsar el botón Aceptar.  Luego el sistema le mostrará un breve mensaje en el que informará del éxito de la actualización.

Recuerde que:
  • El login de usuario no puede ser modificado porque es utilizado para identificarlo al momento de loguearse en la herramienta.
  • El password no se puede visualizar por cuestiones de seguridad, aunque usted lo puede cambiar pero no será posible la recuperación del password anterior.


Deshabilitar

Para deshabilitar un usuario, seleccione de la lista de usuarios registrados el usuario de interés y en el formulario ubicado al lado derecho de la interfaz seleccione en el campo Habilitado la opción No; luego pulse el botón Actualizar. El sistema desplegará una pequeña ventana en donde debe ingresar la razón de la actualizacion de la información del usuario.  Una vez ingresada dicha razón, debe pulsar el botón Aceptar. Luego el sistema le mostrará un breve mensaje en el que informará del éxito de la operación.  Recuerde que los usuarios deshabilitados no podrán ingresar al sistema.
