Manejo de estados


MANUAL DE USUARIO
 TPM-QLabs - ESTADOS DEL EQUIPO


Cuando el usuario selecciona la opción Estado del equipo ubicada en el menú principal, el sistema le desplegará una pantalla como la que se muestra en la Figura 1.  En esta pantalla el usuario podrá crear, visualizar, actualizar, eliminar y restablecer los estados de los equipos o máquinas. Es importante que usted sea consciente de que la información registrada en esta interfaz es parte de la información base del sistema y el hecho de eliminar algún registro puede alterar el funcionamiento normal del sistema.  Por lo tanto, es recomendable que esta pantalla sólo se utilice para agregar y modificar datos.

La información ingresada en esta interfaz es utilizada para definir el estado de un equipo del sistema.
Figura 1. Pantalla de manejo de estados del equipo.
Agregar estado
Para agregar un estado pulse el botón Agregar (1), a continuación diligencie el nombre del nuevo estado en la fila que aparece al inicio de la lista (2) y pulse el botón Guardar (3).  En caso de querer cancelar el proceso de registro de un nuevo estado debe pulsar el botón Cancelar (4).  En la Figura 2 se muestra el proceso a realizar para agregar un nuevo estado.


Figura 2. Pantalla de agregar un nuevo estado.


Actualizar
Para actualizar la información de un estado, debe dar doble clic sobre el estado que desea modificar, luego editar la información sobre el campo de texto ubicado en la columna Nombre (1) y pulsar el botón Guardar.  El sistema desplegará una pequeña ventana donde debe ingresar la razón de la actualización de la información del estado (2).  Una vez ingresada dicha razón, debe pulsar el botón Aceptar (3).  Luego el sistema le mostrará un breve mensaje en el que le informará del éxito de la actualización. En la Figura 3 se muestra la pantalla para llevar a cabo el proceso de actualización de un estado.

Figura 3. Pantalla para modificar un estado.


Eliminar
Para eliminar un estado, seleccione de la lista de estados registrados el estado que desea eliminar (1) y pulse el botón Eliminar (2).  El sistema desplegará una pequeña ventana donde le preguntará si realmente desea eliminar el estado en donde debe seleccionar la opción Si (3), luego el sistema le preguntará la razón por la cual desea eliminarlo, digite la razón y pulse el botón Aceptar.  El sistema le mostrará un breve mensaje en el que le informará del éxito de la eliminación.  En el listado de estados eliminados denotado por una estrella blanca (4) podrá observar el registro que eliminó anteriormente.  En la Figura 4 se muestra la pantalla para llevar a cabo la eliminación de un estado.

Figura 4. Pantalla para eliminar un estado.


Restablecer
Si se ha eliminado un estado, usted tiene la posibilidad de recuperar la información seleccionando la opción para ver los estados eliminados denotada por la estrella blanca (1), luego seleccione de la lista de estados eliminados el estado que desea recuperar (2) y pulse el botón Restablecer (3).  El sistema le preguntará la razón por la cual usted desea restablecer el estado, digite la razón (4) y pulse el botón Aceptar (5).  Luego el sistema le mostrará un breve mensaje en el que le informará sobre el éxito del proceso de restablecimiento.  Si el estado se ha restablecido con éxito, volverá a aparecer en la lista de estados activos denotada por la estrella amarilla (6).  En la Figura 5 se muestra la pantalla para llevar a cabo el proceso de restablecimiento de un estado.

Figura 5. Pantalla para restablecer un estado eliminado.
