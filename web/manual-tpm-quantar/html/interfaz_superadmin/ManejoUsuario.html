MANUAL DE USUARIO
 TPM-QLabs  - MANEJO DE SÚPER USUARIOS


Cuando el usuario selecciona la opción Manejo de superadmin ubicada en el menú principal, el sistema le desplegará una pantalla como la que se muestra en la Figura 1.  En esta pantalla el usuario podrá crear, visualizar, actualizar, habilitar y deshabilitar los usuarios con rol Súper Administrador.
Figura 1. Pantalla para el manejo de manejo de usuarios con perfil Súper Administrador.


Agregar usuario

Para agregar un súper administrador pulse el botón Agregar, diligencie el formulario ubicado al lado derecho de la interfaz para el manejo de super administradores y pulse el botón Guardar.  Los datos del usuario que se deben ingresar para su registro se describen a continuación:
  • Login de usuario: Ingrese el login de usuario  (es un campo obligatorio).
  • Password: Ingrese el password del usuario el cual debe tener como mínimo 8 caracteres (es un campo obligatorio).
  • Repetir password: Repita el password (es un campo obligatorio).
  • Perfil: Seleccione un perfil de usuario.  Un perfil de usuario define las funcionalidades a las cuales tiene acceso cada persona.  Para esta interfaz, el único perfil disponible es el perfil Súper administrador, el cual es un perfil reservado para Quantar S.A.S (es un campo obligatorio).
  • Habilitado: Seleccione si el usuario está habilitado o deshabilitado.  Tenga en cuenta que los usuarios deshabilitados no podrán ingresar al sistema (es un campo obligatorio).


Actualizar

Para actualizar la información de un usuario, seleccione un usuario de la lista de usuarios registrados, actualice la información del usuario haciendo uso del formulario ubicado al lado derecho de la interfaz y pulse el botón Actualizar.  El sistema desplegará una pequeña ventana en donde debe ingresar la razón de la actualización de la información del usuario.  Una vez ingresada dicha razón, debe pulsar el botón Aceptar.  Luego el sistema le mostrará un breve mensaje en el que le informará del éxito de la actualización.


Deshabilitar

Para deshabilitar un usuario, seleccione de la lista de usuarios registrados el usuario de interés y en el formulario ubicado al lado derecho de la interfaz seleccione en el campo Habilitado la opción No; luego pulse el botón Actualizar.  El sistema desplegará una pequeña ventana en donde debe ingresar la razón de la actualizacion de la información del usuario.  Una vez ingresada dicha razón, debe pulsar el botón Aceptar.  Luego el sistema le mostrará un breve mensaje en el que informará del éxito de la operación.  Recuerde que los usuarios deshabilitados no podrán ingresar al sistema.
